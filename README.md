# WebTech-2

# How to run:
- Step 1: Clone the project
- Step 3: Rename example.env(in server folder) as .env and set it up (according to your local variables)
- Step 2: Open two terminals and enter the commands one by one
    - Terminal 1:
      ~~~bash
      cd server
      npm i
      npm run dev
      ~~~
    - Terminal 2:
      ~~~bash
      cd client
      npm i
      npm start
      ~~~
- Step 4: Enjoy!!
